Mobile Banking UX Redesign

모바일 뱅킹 송금 과정을 더 빠르고 직관적으로 개선하는 UX 프로젝트

Project Overview

기존 모바일 뱅킹 앱의 송금 과정은 단계가 많고 사용자가 혼란을 느끼는 문제가 있었습니다. 이 프로젝트는 송금 과정을 단순화하고 사용자의 오류 가능성을 줄이는 것을 목표로 진행되었습니다.

Problem

사용자는 송금 과정에서 여러 단계를 거쳐야 했으며, 각 단계에서 입력 오류가 자주 발생했습니다. 또한 전체 과정이 직관적이지 않아 사용 시간이 길어지는 문제가 있었습니다.

Research

사용자 인터뷰와 기존 서비스 분석을 통해 주요 문제점을 파악했습니다.

  • 송금 과정 단계가 너무 많음
  • 입력 필드 구조가 복잡함
  • 확인 과정이 불명확함

Solution

송금 과정을 5단계에서 3단계로 줄이고, 입력 UI를 단순화했습니다. 또한 확인 단계를 명확하게 디자인하여 사용자 실수를 줄였습니다.

Result

  • 송금 완료 시간 30% 감소
  • 입력 오류 40% 감소
  • 사용자 만족도 향상
